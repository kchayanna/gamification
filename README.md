# gamification
Бэкенд для веб-приложения, которое превращает выполнение целей и задач в игру с системой опыта (XP), прогрессом и «дорогами» (ветками развития). 
Идея проекта:
Пользователь создаёт цели (ветки), разбивает их на чекпоинты (задачи) и получает опыт (XP) за выполнение.
Примеры:
получить права → теория / вождение
подготовка к ЕГЭ → ежедневные задания
Прогресс визуализируется как движение по дороге

Технологии:
FastAPI — backend API
Python — основной ЯП
JSON (файловая база) — хранение данных (MVP)
Pydantic — валидация данных
CORS Middleware — подключение frontend

Сейчас/пока:
используется JSON вместо базы данных
отсутствует авторизация
один пользователь

идеи развития:
1) совместные цели(можно добавлять друзей и соревноваться
2) партнеры/награды
3) отдельный режим для визуализациии структуры задач и целей(подробнее в обновлении)
4) умный ввод
